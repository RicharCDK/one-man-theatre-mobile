# Театр одного актора

В межах 5 лабораторної роботи розроблено сайт на тему театра. 

Раніше створени макет адаптовано під мобільні пристрої.

Вже на основі адаптованого макету створено три сторінки: головна, "Про нас" та "Афіша".
На сторінці "Афіша" реалізовано інтерактивні фільтри, які дозволяють дозволяють шукати серед потрібних вистав.
